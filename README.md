# TrendSignal

TrendSignal is a personal research tool that discovers emerging consumer product trends across search, video, marketplace, and social data sources.

The platform identifies accelerating product-level demand signals, validates them across multiple sources, and maps trends to potential public-company beneficiaries for further investigation.

TrendSignal is intended as a research screener only. It does not provide investment advice or execute trades.

## Current Data Sources

- Google Trends
- YouTube Data API v3
- Reddit API (pending)
- eBay API (pending)
- Etsy API (pending)

## Objective

Discover unexpected consumer products and behaviors early, validate demand across multiple independent sources, and surface potentially investable opportunities for manual research.
